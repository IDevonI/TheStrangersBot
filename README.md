# RoomServiceBot

Jest to bot napisany w języku Java przy użyciu JDA (Java Discord API).
Stworzony na potrzeby serwera discord The Strangers.
Wprowadza do użytku polecenia '!embed' do tworzenia osadzonych postów 
oraz polecenie '!stangers sendmsg' do wysyłania osadzonych wiadomości na kanałach serwera.

This is a bot written in Java language by using JDA (Java Discord API).
Created for The Strangers discord server use purpose.
It introduces '!embed' command, which creates embed posts, and '!strangers sendmsg' command, 
which allows to send embed messages on server channels.
